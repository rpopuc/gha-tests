Isso é um exemplo de uso do GitHub Actions.

O GitHub Actions é um serviço de integração contínua (CI) que permite automatizar o fluxo de trabalho do desenvolvimento de software com base em eventos do GitHub.

Com o GitHub Actions, você pode criar tarefas personalizadas para qualquer tipo de trabalho que você possa imaginar, e configurá-las para serem executadas sempre que houver uma solicitação de push para o repositório do GitHub.

O GitHub Actions combina os recursos de vários serviços de integração contínua, incluindo o Travis CI e o Azure Pipelines, para fornecer uma experiência de integração contínua flexível e integrada no GitHub.
